2. **Claude托管代理功能上线。** 呼应此前提到的 **Agent 转型** 战略 ✨，Claude 平台发布了托管代理。 开发者现在可以在 [Claude代理平台(AI资讯)](http://claude.com/blog/new-in-claude-managed-agents) 创建自动化智能体。 控制台支持调用 these **托管服务** 🛠️ 显著提升效率。 该功能简化了 **AI 部署流程** 使其更加灵活。 智能助手领域由此开启了全新的技术篇章 (o_o)。<br/>

3. **Anthropic携手SpaceX上调限额** Anthropic 官宣大幅上调其 **Claude使用限制** 🚀 啦。[SpaceX算力合作(AI资讯)](https://www.anthropic.com/news/higher-limits-spacex) 将提供极其 **庞大算力** 支持。这让 **Pro用户** 能在复杂任务中拥有更多 **对话次数** (๑•̀ㅂ•́)و✧。这次联手标志着 **AI基建** 领域竞争进入了 **全新阶段**。[查看具体额度提升规则](https://x.com/claudeai/status/2052060696255283346) 现已在 **社交媒体** 公开。

4. **ChatGPT 进驻 Excel 套件。** 搭载最强 **GPT-5.5** 内核的办公助手现已登陆表格软件 🤯。 它能帮用户编写 **复杂公式** 或者解释数据逻辑。 这项 [办公套件集成方案(AI资讯)](https://chatgpt.com/apps/spreadsheets/) 能够极速分析表格。 社交媒体上关于 [办公助手新插件](https://x.com/gdb/status/2051866486658994636) 的讨论热度极高。 广大办公族以后大概能少加点班了 (^_−)☆。<br/><video src="https://source.hubtoday.app/images/2026/05/news_01kqzzzveqext9nyds8enkxb3d.mp4" controls="controls" width="100%"></video><br/>

### 前沿研究

1. **学术团队打破搜索垄断。** **搜索新秀** OpenSeeker-v2 凭借学术成果登顶榜单 🌟。 该团队仅靠 [SFT 训练方法(AI资讯)](https://www.aibase.com/zh/news/27704) 便打破了行业巨头垄断。 训练过程仅需 **万条数据** 🚀 即可超越大厂性能。 详细的技术突破可见其 [学术团队前沿论文](https://arxiv.org/pdf/2605.04036) 中的描述。 目前该模型已经 **正式开源** 方便开发者使用 (⊙o⊙)。

2. **农业专用模型 CropVLM 发布。** 研究人员发布了 [农业专用大模型(AI资讯)](https://arxiv.org/abs/2605.03259) 用于作物分析 🌿。 该模型通过 **语义对齐** 掌握了三十余种作物细节。 配合 GitHub 上的 [HOS-Net 框架](https://github.com/boudiafA/CropVLM) 可搞定零样本检测。 其 **分类准确率** 达到了惊人的七成以上 (๑•̀ㅂ•́)و✧。 自动表型分析技术将变得更加简单高效 🌾。

3. **ClawMark 代理评测基准。** 专门针对 **智能同事模型** 发布的评测体系震撼登场 🚀。 传统静态评估已无法满足 [动态办公场景(AI资讯)](https://arxiv.org/abs/2604.23781) 的测试需求。 项目涵盖百项 **专业任务** 并通过脚本进行客观评分。 目前主流模型在长流程中的 **成功率** 仅为两成 (ง •_•)ง。 适应性短板成为了当前 **代理智能体** 发展的核心痛点。<br/>

4. **AniMatrix 动漫生成模型。** 研究人员推出的 [AniMatrix模型(AI资讯)](https://arxiv.org/abs/2605.03652) 追求极致艺术表达。 该模型 🎬 摒弃了僵硬的物理规律让画面更灵动。 系统利用 **AniCaption** 技术智能提取各类制作变量。 其在艺术运动得分方面远超同类 **顶尖模型** ヾ(≧▽≦*)o。 团队承诺近期将向全球开源模型权重。 <br/>

5. **微软发布代理可解释性模型。** 微软研究院提出了 [全新的可解释性(AI资讯)](https://arxiv.org/abs/2605.03808) 框架用于智能体研究。 代理模型 🤖 能够自主迭代出精确且可读的 **回归器** 结果。 小模型只需读取字符串即可完成 **精准预测** (｡•̀ᴗ-)。 该方案在数十个数据集上大幅超越了传统统计模型。 它在 BLADE 榜单中将性能提升到了极致 🔝。<br/>![AI资讯：微软Agentic-imodels自动化研究循环架构图](https://source.hubtoday.app/images/2026/05/news_01kqzzy4z8ext9nydk93sbch56.avif)<br/>
